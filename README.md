# Website Eminem

Este é um projeto de site completo sobre o artista **Eminem**, construído com HTML, CSS e JavaScript como parte dos meus estudos em desenvolvimento web.

O objetivo é praticar a criação de sites responsivos com múltiplas páginas e componentes visuais, aplicando semântica HTML, estilos avançados em CSS e pequenos comportamentos em JavaScript.

---

## Conteúdo do repositório

O site possui as seguintes páginas:

| Arquivo | Descrição |
|--------|-----------|
| `index.html` | Página inicial |
| `discografia.html` | Discografia do artista |
| `galeria.html` | Galeria de imagens relacionadas ao Eminem |
| `premios.html` | Lista de prêmios e conquistas |
| `formulario.html` | Página de formulário (exemplo de interatividade) |
| `relacionados&fontes.html` | Página de links e referências |
| `style.css` | Estilos utilizados no site |
| `button.css` | Estilos adicionais de botões |
| `button.js` | Script JavaScript usado no site |
| `imagens/` | Imagens utilizadas no projeto |
